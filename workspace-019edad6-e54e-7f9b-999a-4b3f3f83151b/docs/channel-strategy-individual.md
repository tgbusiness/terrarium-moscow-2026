# Стратегия продвижения по каналам (Частный специалист, Москва 2026)

## 1. Экосистема каналов и воронка продаж

```
[ Авито ] [ Профи.ру ] [ Яндекс Услуги / Карты ] [ Соцсети / VK ]
    │          │                 │                      │
    └──────────┴────────┬────────┴──────────────────────┘
                        ▼
           [ Одностраничный Лендинг ]  <─── (Контекст / Прямые ссылки)
                        │
                        ▼
          [ Telegram Бот (Прием заявок) ]
                        │
                        ▼
      [ Квалификация мастера -> Предоплата -> Заказ ]
```

## 2. Разбор ролей каждого канала

### 2.1 Авито (Основной генератор заявок)
* **Роль:** Охват «горячего» спроса. Люди уже ищут «террариум купить», «террариум на заказ», «террариум для эублефара».
* **Формат присутствия:** Профессиональный профиль частного мастера с подтвержденными документами («Паспорт проверен»).
* **Стратегия:** 
  1. Создать 5–10 оптимизированных объявлений (разбивка по животным: для эублефаров, для агам, для змей, для хамелеонов + изготовление по размерам).
  2. Использовать качественные инфографичные обложки.
  3. В описании активно закрывать на запрос каталога/расчета стоимости в чате.
* **Целевое действие:** Написать в чат Авито габариты желаемого террариума или вид питомца.

### 2.2 Профи.ру (Profi)
* **Роль:** Получение высокочековых индивидуальных заказов на изготовление сложных конструкций и стоек.
* **Формат присутствия:** Заполненная анкета специалиста, сданные тесты (если применимо `[VERIFY-2026]`), детальный прайс-лист по видам работ (резка, склейка, электрика, фон).
* **Стратегия:** Мгновенный персональный отклик на публикуемые заказы с четким расчетом цены и приложением фото похожих готовых работ.
* **Целевое действие:** Созвон с клиентом, выход на замер или согласование чертежа.

### 2.3 Яндекс (Услуги / Карты / Бизнес)
* **Роль:** Локальное доверие и сбор гео-трафика (Москва и МО).
* **Формат присутствия:** Подтвержденная карточка в Яндекс Бизнесе («Частная мастерская террариумов Имя Фамилия»). 
* **Стратегия:** Максимальный сбор отзывов от довольных клиентов. Добавление товаров и услуг с ценами («Террариум 60х40х40 — от ХХХХ руб.»). Включение бесплатного или базового платного продвижения на Картах `[VERIFY-2026]`.
* **Целевое действие:** Переход на Лендинг или звонок.

### 2.4 Социальные сети (VK Паблик + Telegram Канал)
* **Роль:** Прогрев сомневающихся, демонстрация экспертности, формирование сообщества.
* **Контент:**
  * Видео-рилсы/клипы: «Как я клею идеальный шов», «Ошибки вентиляции в террариумах», «Запуск водопада в палюдариуме».
  * До/После: пустая банка vs полностью оформленный биоактивный террариум.
  * Отзывы клиентов и фото счастливых рептилий в новых домах.
* **Целевое действие:** Подписка -> обращение в Telegram-бот для расчета.

### 2.5 Одностраничный Лендинг
* **Роль:** Визитная карточка премиум-уровня. Внушает максимальное доверие, структурированно отвечает на все вопросы, отсекает нецелевой спам.
* **Трафик:** Переходы из соцсетей, ссылка в описаниях профилей, точечная Яндекс Директ реклама (по низкочастотным запросам).
* **Целевое действие:** Заполнение квиз-формы или заявка на обратный звонок/мессенджер.
