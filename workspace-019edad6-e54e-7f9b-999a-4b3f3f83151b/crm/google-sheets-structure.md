# Структура колонок Google Sheets / Airtable (google-sheets-structure.md)

Для частного специалиста в Москве, который не хочет администрировать SQL-базы данных, идеальным решением в 2026 году остается бесплатная таблица Google Sheets или Airtable.

Создайте новый документ Google Sheets и назовите первую вкладку **`Leads & Orders`**. Настройте 16 колонок строго по таблице ниже:

| № | Название колонки | Тип / Формат | Пример заполнения | Зачем нужно |
| :-| :--- | :--- | :--- | :--- |
| A | **ID Заказа** | Текст | `L-123456` или `ORD-2026-001` | Уникальный номер для поиска и привязки |
| B | **Дата заявки** | Дата / Время | `18.06.2026 14:30` | Контроль скорости ответа (правило золотого часа) |
| C | **Источник** | Выпадающий список | `Landing` / `Avito` / `Profi` | Аналитика эффективности каналов |
| D | **Имя клиента** | Текст | `Алексей` | Персональное общение |
| E | **Контакты (Телефон)** | Текст | `+79991112233` | Прямая связь с клиентом |
| F | **Способ связи** | Выпадающий список | `telegram` / `whatsapp` / `phone`| Выбор мессенджера |
| G | **Питомец** | Текст | `Бородатая агама` | Понимание биологии и вентиляции |
| H | **Размеры (ДхШхВ)** | Текст | `80 х 40 х 40 см` | Раскрой стекла и чертеж |
| I | **Пакет / Тип** | Выпадающий список | `Комплекс Под Ключ` | Определение состава работ |
| J | **Пожелания / 3D Фон**| Текст | `3D-фон под скалу, УФ-свет`| Учет авторского декора |
| K | **СТАТУС** | Выпадающий список | `🟣 IN_PRODUCTION` | **Главная колонка воронки (Status)** |
| L | **Общая Цена (руб.)**| Валюта | `14500` | Финансовый учет сделки |
| M | **Предоплата (руб.)**| Валюта | `7000` | Контроль полученного аванса |
| N | **Дедлайн сдачи** | Дата | `25.06.2026` | Производственный календарь |
| O | **Адрес / Доставка** | Текст | `Самовывоз` или `ул. Ленина, д.5`| Организация логистики |
| P | **Технические заметки**| Текст | `Черный герметик, замок на дверцы`| Памятка по сборке |

## 🎨 Советы по цветовому оформлению (Условное форматирование)
Для удобства визуального контроля настройте условное форматирование для колонки **`K` (СТАТУС)**:
- `🟡 NEW_LEAD` — светло-желтый фон (новая заявка).
- `🟠 ESTIMATING` — светло-оранжевый фон (составление сметы).
- `🔵 PREPAYMENT_RECEIVED` — голубой фон (аванс получен, резка стекла).
- `🟣 IN_PRODUCTION` — фиолетовый фон (сборка, лепка 3D-фона).
- `🟢 READY_FOR_DELIVERY` — светло-зеленый фон (террариум готов, отснят отчет).
- `🏁 COMPLETED` — серый фон с зачеркнутым текстом (выдан, оплачен).
- `🔴 CANCELLED` — красный фон (отказ / отмена).
